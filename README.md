# 🏠 租房对比神器 · Rent Radar

按你的偏好为每套候选房源**打分排序**，并算清每套房的**真实开销**——把通勤时间按你的时薪折算成钱，让「便宜但远」的房子现出原形，帮你理性做决定。

> 纯前端应用，所有数据只存在你自己的浏览器里，不上传任何服务器。AI 分析使用你自己的 API Key。

## ✨ 特性

- **多维度加权打分**：通勤、价格、采光、独立性、空间，按你「最看重什么」自定义权重。
- **真实开销计算**：`真实月成本 = 房租 + 月交通费 + 通勤时间折算成本`。这是本工具的灵魂——很多人忽略了通勤时间的隐形成本。
- **公式打分 + AI 解释分离**：打分用确定性公式（结果稳定、可复现），大模型只负责用「人话」点评权衡、给建议，不参与算分。
- **自带模型，零成本托管**：填你自己的 API Key，兼容 OpenAI 格式（DeepSeek / 通义千问 / OpenAI / 本地 Ollama 均可）。
- **本地存储**：输入自动保存，刷新不丢。


`dist/` 是纯静态文件，可直接部署到 GitHub Pages、Vercel、Netlify 等。

## 🧮 打分逻辑

1. 每个维度统一为「值越大越好」，在所有房源间做 min-max 归一化到 0-100。
2. 用户权重归一化后加权求和，得到 0-100 的总分。
3. 真实开销中的通勤时间成本：`时薪 × 每天通勤小时 × 工作天数`，时薪 = `月收入 / (工作天数 × 8)`。

详见 [`src/lib/scoring.ts`](src/lib/scoring.ts)。
