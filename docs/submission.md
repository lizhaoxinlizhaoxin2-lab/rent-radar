## 项目名称
租房对比神器（Rent Radar）

## 一句话简介（≤30 字）
把通勤时间折算成钱，帮你算清哪套房真的更划算。

## 副标题 / Slogan
别再凭感觉选房——可解释、可复现的租房决策工具。

---

## 详细介绍（长描述）

**租房是大多数人最频繁的高金额决策之一，却几乎全靠拍脑袋。**

价格、通勤、采光、独立性、空间……每个人在意的点都不一样；更要命的是，多数人只盯着月租，忽略了**通勤时间这笔最大的隐形成本**——一套月租便宜 500 的房子，如果每天多花 1 小时通勤，按你的时薪折算，每月反而可能更贵。

「租房对比神器」用一个**多维度加权打分模型**解决这个问题：

- **你只需表达「最看重什么」**：每个维度独立选在意程度（6 档，每档有描述）。
- **工具替你算清真实开销**：房租 + 杂费（取暖/网费/物业）+ 交通 + **通勤时间折现**。其中「通勤 1 小时值多少钱」还能自己调——上班轻松调低、讨厌通勤调高，告别一刀切。
- **结果可解释、可复现**：打分用确定性公式，不是让大模型「拍」一个分数；大模型只负责用人话点评权衡、给建议。
- **可视化对比**：分数条、雷达图、奖牌排名，一眼看清谁更适合你。
- **链接知乎**：输入城市/区域，一键查知乎上的真实居住口碑与避坑帖，把「定房前先上知乎做功课」做进流程里。

纯前端实现，数据只存浏览器本地，隐私友好；AI 分析用用户自己的 API Key（兼容 DeepSeek / 通义 / OpenAI / 本地 Ollama）。

## 核心亮点（3 条，用于卡片展示）
1. 独创「通勤时间折现 + 可调时间价值」，把隐形成本算成真金白银。
2. 公式打分 + AI 解释分离，结果稳定可复现，不是玄学。
3. 纯前端、零后端成本、隐私优先，AI 用自己的 Key 即开即用。
